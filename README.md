# MEDICAL-DATA-ANALYSIS
Hospital Medical Data Dashboard

1. Project Title / Headline

🏥 Hospital Medical Data Dashboard – Power BI Analytics Report

An interactive Power BI dashboard designed to analyze hospital operations, patient records, doctor specialization data, and healthcare trends through dynamic visualizations and KPI monitoring.

2. Short Description / Purpose

This dashboard provides a complete overview of hospital and patient-related analytics, including admissions, diagnosis trends, doctor availability, patient demographics, allergies, and treatment insights. It helps healthcare organizations monitor operational performance and make informed decisions using data-driven analysis.

The dashboard is designed for hospital administrators, healthcare analysts, and medical management teams to improve efficiency and patient care monitoring.

3. Tech Stack

The dashboard was built using the following tools and technologies:

📊 Power BI Desktop – Main reporting and dashboard visualization platform
🔄 Power Query – Data cleaning, transformation, and preprocessing
🧠 DAX (Data Analysis Expressions) – Used for KPIs, measures, and calculated insights
🔗 Data Modeling – Relationship building between hospital datasets
🗂️ Excel / CSV Dataset – Used as the primary healthcare data source
📁 File Format – .pbix for dashboard development and .png for previews
4. Data Source
Source:

Healthcare and hospital management dataset containing:

Patient records and admissions
Doctor specialization details
Diagnosis and treatment data
Patient demographics
Allergies and medical conditions
Weight, height, and health metrics
Location and city-wise patient distribution

The dataset is structured to simulate real-world hospital operations and patient healthcare analytics.

5. Features / Highlights
📌 Business Problem

Hospitals generate large volumes of patient and operational data daily, making it difficult to manually track admissions, diagnoses, doctor availability, and patient treatment patterns efficiently.

🎯 Goal of the Dashboard

The main objectives of this dashboard are to:

Monitor patient admissions and discharge trends
Analyze diagnosis and treatment patterns
Track doctor specialization and availability
Understand patient demographics and health metrics
Identify city-wise patient distribution
Improve healthcare decision-making using visual analytics

8. Key Insights
Congestive Heart Failure and Appendicitis were among the most common diagnoses
Patient admissions varied significantly across months and days
Certain cities showed higher patient concentration compared to others
Male and female patient distribution was relatively balanced
Same-day discharge tracking helped monitor hospital operational efficiency
Doctor specialization data provided better workforce visibility
9. Conclusion

This Hospital Medical Data Dashboard converts complex healthcare datasets into actionable insights using Power BI visualizations and analytical reporting. It enables hospitals and healthcare professionals to monitor patient trends, operational efficiency, and medical performance effectively, supporting smarter healthcare management and decision-making.
